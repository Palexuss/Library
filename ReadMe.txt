Реализация проекта "Библиотека"
Технологии: БД(SQL: h2), Spring, Maven, JUnit, Java 8 (Проверялось на Intelij IDEA 2020.3.2 Ultimate)


Запуск:
1. Запуск производится в классе LibraryApplication;
2. Проверка работоспособности большинства методов контроллера производится в LibraryApplicationTests;
3. Для проверки команд контроллера перейдите по ссылке http://localhost:8090/{запросы контроллера} 
(Пример: http://localhost:8090/books/list для проверки созданных книг)
4. ПРОБЛЕМЫ РЕАЛИЗАЦИИ ПРОЕКТА: попытался реализовать Many-to-Many для связи книг и авторов дабы избежать
создания повторяющихся экземпляров классов, но возникли определенные 
проблемы реализации УДАЛЕНИЯ КНИГ и ВЫВОДА СПИСКА АВТОРОВ КНИГ, поэтому для проверки этих методов
необходимо использовать SQL запросы в h2 console (http://localhost:8090/h2-console/ USERNAME: ss)
